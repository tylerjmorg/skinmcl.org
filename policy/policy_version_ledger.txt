# Policy Version Ledger


Starting 2025-10-11, all policies will now include an ID and SRI
that updates automatically as changes are made. Policies refere-
nced must always match the SRI and ID algorithm in `/access_inf-
o.js` to this ledger or else it is not legitimate policy or has
been tempered with.

This text markdown file holds the official ledger with the IDs
and SRIs that are recognized as legitimate.
